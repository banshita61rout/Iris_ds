#  Iris Flower Classification

A Machine Learning project that classifies Iris flowers into three species using their sepal and petal measurements. The project includes Exploratory Data Analysis (EDA), data preprocessing, and multiple classification models.

## Dataset

- **Samples:** 150
- **Features:** 4 Numerical Features
- **Target:** Species (Setosa, Versicolor, Virginica)

## Exploratory Data Analysis

Visualizations performed:
- Count Plot
- Pair Plot
- Histogram
- Scatter Plot
- Box Plot
- Violin Plot
- Correlation Heatmap
- Pie Chart

## Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report
- Model Comparison
